# EvilPi, Setup eines Man-In-The-Middle Raspberry Pi

## Übersicht
### Setup
* Raspberry Pi 3, Internetzugriff über Ethernet
### Ergebnis
* Ein WLAN Access Point, sodass Netzwerkverkehr mitgeschnitten werden kann.
